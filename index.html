Guide des contrats

À quoi sert
chaque contrat ?

Cliquez sur un contrat pour voir son utilité, le bon moment pour l'utiliser, et les textes de loi applicables.

Ce que c'est
Un engagement contractuel — mutuel ou unilatéral — par lequel une ou deux parties s'obligent à ne pas divulguer des informations confidentielles reçues dans le cadre d'une relation professionnelle. Il peut couvrir des données techniques, financières, commerciales ou stratégiques.
Quand l'utiliser
Avant toute négociation commerciale, discussion de partenariat, levée de fonds, entretien avec un investisseur, ou partage d'informations sensibles avec un prestataire, sous-traitant ou candidat à un poste stratégique.
Ce qu'il protège
Savoir-faire, données clients, plans de développement, prototypes, informations financières non publiées, stratégie commerciale, code source, formules ou procédés techniques.
Durée typique
De 1 à 5 ans selon les enjeux. Certaines clauses peuvent être à durée indéterminée pour les secrets commerciaux les plus sensibles.
Art. 1103 Code civil Art. L151-1 Code de commerce Loi du 30 juil. 2018 — secret des affaires Directive UE 2016/943
Ce que c'est
Un contrat par lequel un prestataire (freelance, consultant ou société) s'engage à réaliser une mission définie pour un client, contre rémunération. Il précise les livrables, les délais, le prix, les conditions de paiement et la propriété intellectuelle.
Quand l'utiliser
Dès qu'une mission est confiée à un prestataire externe : développement web, conseil en stratégie, design graphique, rédaction, formation, audit, etc. Indispensable même pour les petites missions.
Points clés couverts
Description de la mission, livrables attendus, calendrier, prix HT/TTC, conditions de paiement, pénalités de retard, propriété intellectuelle, clause de confidentialité et résiliation.
Risque sans contrat
Litige sur les livrables, non-paiement, revendication de propriété intellectuelle par le client, ou qualification en contrat de travail déguisé (risque de requalification en CDI).
Art. 1710 Code civil Art. L441-1 Code de commerce Art. L111-1 Code de la propriété intellectuelle
Ce que c'est
Le document qui régit les relations entre un vendeur et ses clients : prix, délais de livraison, politique de retour, responsabilités et litiges.
Quand l'utiliser
Obligatoires pour tout e-commerçant vendant à des particuliers (B2C). Fortement recommandées en B2B pour sécuriser chaque transaction.
Art. L441-1 Code de commerce Art. L221-11 Code de la consommation Directive UE 2011/83
Ce que c'est
Un accord confidentiel entre associés qui complète les statuts : gouvernance, droits de vote, cession de parts, clauses de sortie et de préemption.
Quand l'utiliser
À la création d'une société à plusieurs associés, ou lors d'une levée de fonds. Indispensable pour prévenir les conflits entre fondateurs.
Art. 1134 Code civil Art. L227-1 Code de commerce
Ce que c'est
Le contrat qui formalise la relation employeur-salarié : poste, rémunération, durée de travail, période d'essai et conditions de rupture.
Quand l'utiliser
Dès votre premier recrutement. Le CDI est la règle générale, le CDD est l'exception réservée à des cas précis (remplacement, accroissement temporaire d'activité).
Art. L1221-1 Code du travail Art. L1242-1 Code du travail
Ce que c'est
Le document fondateur de votre société : objet social, capital, gouvernance, répartition des parts et règles de fonctionnement interne.
Quand l'utiliser
Obligatoires pour immatriculer votre société au RCS. La SAS offre plus de flexibilité pour les startups, la SARL est plus encadrée et rassurante pour les PME.
Art. L227-1 Code de commerce Art. L223-1 Code de commerce
